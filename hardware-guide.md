| 位号 | 说明 |
| :--- | :--- |
| J1 | MCU JTAG接口 |
| J2 | TF卡插座 |
| J4 | Wi-Fi陶瓷天线 |
| J5 | ARM调试串口（板载USB UART） |
| J6 | MCU调试串口（板载USB UART） |
| J9 | 扩展接口（40脚，兼容树莓派扩展板） |
| J10 | USB Device |
| J13 | USB Host |
