### 接口说明

| 位号 | 说明 |
| :--- | :--- |
| J1 | MCU JTAG接口 |
| J2 | TF卡插座 |
| J4 | Wi-Fi陶瓷天线 |
| J5 | ARM调试串口（板载USB UART） |
| J6 | MCU调试串口（板载USB UART） |
| J9 | 扩展接口（40脚，兼容树莓派扩展板） |
| J10 | USB Device |
| J13 | USB Host |
| J14 | PCM OUT的MCLK的测试脚 |
| J23 | 3.5mm 音频输出接口 |
| J30 | 静音跳线 |
| J34 | DC 5V输入 |
| D2 | 电源输入指示 |
| D5 | 功能指示 |
| K1 | 复位按钮 |
